Development Environments

It is always important to find a text editor that suits your needs and provides all the capabilities you look for when programming. While there are many text editors available, these are a few of our favorites. The portable versions do not require administrative privileges, and will work on the school computers.

Multiple language support, modern editing tools, automated script handling, GIT integration, extensions.

Full Portable

Atom

Multiple language support, modern editing tools, automated script handling, GIT integration, extensions.

Full Portable

Sublime Text

Multiple language support, modern editing tools, automated script handling, GIT integration, extensions.

Full Portable

Brackets

Web support only, amazing JavaScript support, live web previews, GIT integration, extensions.

Full Portable

Common CSS Libraries

Working with native CSS from scratch can be annoying. Nowadays, many front end developers use either Sass or 'CssinJS', programming languages that compile to raw CSS. Common CSS libraries circulating the web are actually built using these langauges. They've already done the hardwork for you! Why not use them in your own projects?
